# 🐍 PipBot — Умный модератор для Telegram

> 🛡️ **"PipBot: Чистота в чате — наша работа"**

**PipBot** — современный Telegram-бот для модерации групп, разработанный на Python с использованием aiogram 3.x. Конкурент Group Help и ChatKeeperBot.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🚫 Антимат | Удаление нецензурной лексики (RU, EN, UA, KZ) |
| 🔁 Антифлуд | Защита от частых сообщений |
| 🔗 Антиссылки | Блокировка ссылок (белый список доменов) |
| 📢 Антикапс | Удаление сообщений с >70% заглавных букв |
| 👥 Антиупоминания | Блокировка массовых @упоминаний |
| 😀 Антиэмодзи | Ограничение количества эмодзи |
| ⚠️ Предупреждения | Система 3 warn = мут/бан |
| 🧩 Капча | Математическая задача при входе |
| 👋 Приветствие | Кастомный текст с переменной {name} |
| 📊 Статистика | Нарушения, топ активных, топ нарушителей |
| 📁 Экспорт | Выгрузка статистики в CSV |
| 💰 Pips | Внутренняя валюта за активность |
| 🛡️ PipShield | Защита от мута за Pips |
| ⭐ Репутация | Рейтинг участников |
| 📅 Отчёты | Ежедневные отчёты администраторам |
| 📝 Логи | Все действия в отдельный чат |

## 🚀 Быстрый старт

### 1. Клонировать и установить

```bash
git clone https://github.com/yourusername/pipbot.git
cd pipbot
pip install -r requirements.txt
```

### 2. Настройка

```bash
cp .env.example .env
```

Открой `.env` и вставь свой токен:

```env
BOT_TOKEN=123456789:AABBCCDDEEFFGGHHIIJJKKLLMMNNOOPPQQRRss
SUPER_ADMINS=ваш_telegram_id
```

### 3. Запуск

```bash
python bot.py
```

## ⚙️ Конфигурация (.env)

| Переменная | Описание | По умолчанию |
|-----------|----------|--------------|
| `BOT_TOKEN` | Токен бота от @BotFather | — |
| `SUPER_ADMINS` | ID супер-админов через запятую | — |
| `DATABASE_URL` | URL базы данных | `sqlite+aiosqlite:///pipbot.db` |
| `USE_REDIS` | Использовать Redis | `false` |
| `USE_OPENAI` | Использовать ChatGPT для токсичности | `false` |
| `OPENAI_API_KEY` | API ключ OpenAI | — |
| `DEFAULT_LANG` | Язык по умолчанию | `ru` |
| `DAILY_REPORT_TIME` | Время отчёта (HH:MM) | `09:00` |

## 📋 Команды

### Для всех участников

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/help` | Список команд |
| `/rules` | Правила группы |
| `/myprofile` | Мой профиль (Pips, репутация, предупреждения) |
| `/shop` | Магазин Pips |
| `/report` | Пожаловаться (ответом на сообщение) |

### Для администраторов

| Команда | Описание |
|---------|----------|
| `/warn [причина]` | Предупреждение (ответом) |
| `/unwarn` | Снять предупреждение (ответом) |
| `/mute [время] [причина]` | Заглушить (пример: `/mute 30m спам`) |
| `/unmute` | Разглушить (ответом) |
| `/ban [причина]` | Заблокировать (ответом) |
| `/unban` | Разблокировать (ответом) |
| `/stats` | Статистика нарушений |
| `/logs` | Последние действия модерации |
| `/settings` | Настройки группы |
| `/setwelcome текст` | Изменить приветствие |
| `/setfarewell текст` | Изменить прощание |
| `/setrules текст` | Установить правила |
| `/setlogchat -100XXXXXXX` | Установить чат для логов |
| `/badword добавить слово` | Добавить слово в чёрный список |
| `/badword удалить слово` | Удалить слово из чёрного списка |
| `/allowdomain домен.com` | Добавить домен в белый список |
| `/removedomain домен.com` | Удалить домен из белого списка |

## 💰 Система Pips

Pips — внутренняя валюта PipBot:
- **+1 Pip** за каждое сообщение
- **+1 Репутация** за каждые 10 сообщений
- **-1 Репутация** за нарушения

### Магазин (`/shop`):
- 🛡️ **PipShield** — 50 Pips — защита от следующего мута
- ⭐ **+1 Репутация** — 30 Pips

## 🚀 Деплой

### Railway / Render

1. Подключи GitHub репозиторий
2. Добавь переменные окружения из `.env.example`
3. Команда запуска: `python bot.py`

### PythonAnywhere

```bash
pip install -r requirements.txt --user
python bot.py
```

### Docker

```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
CMD ["python", "bot.py"]
```

## 🤝 Добавить бота в группу

1. Добавь @YourPipBot в группу
2. Назначь его **администратором** со следующими правами:
   - ✅ Удалять сообщения
   - ✅ Блокировать участников
   - ✅ Ограничивать участников
3. Открой `/settings` для настройки
